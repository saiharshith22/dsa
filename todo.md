## **Phase 1: Foundation Building (Weeks 1-3)**

### **Step 1: Master Programming Fundamentals**
Since you're using JavaScript, ensure you're comfortable with:
- Variables, loops, conditionals, functions
- Time & Space complexity (Big-O notation)[5]
- Problem-solving approach and pattern recognition

### **Step 2: Basic Data Structures**[2][6]

#### **Week 1: Arrays & Strings**
**Concepts:**
- Array manipulation, traversal, 2D arrays
- String manipulation, palindromes, anagrams
- Two-pointer technique
- Sliding window pattern

**Practice:** 15-20 problems
- LeetCode: Two Sum, Best Time to Buy/Sell Stock, Valid Palindrome, Longest Substring Without Repeating Characters

***

#### **Week 2: Hashing (Maps & Sets)**
**Concepts:**
- Hash tables/maps, hash sets
- Frequency counting, lookups
- When to use hashing vs arrays[2]

**Practice:** 10-15 problems
- LeetCode: Contains Duplicate, Group Anagrams, Subarray Sum Equals K

***

#### **Week 3: Mathematics & Bit Manipulation**
**Concepts:**
- Prime numbers, GCD/LCM, factorial
- Bitwise operators, XOR tricks
- Fast exponentiation

**Practice:** 10 problems
- LeetCode: Power of Two, Single Number, Counting Bits

***

## **Phase 3: Non-Linear Data Structures (Weeks 7-10)**

#### **Week 7-8: Trees**[6][2]
**Concepts:**
- Binary trees, BST properties
- Tree traversals (inorder, preorder, postorder, level-order)
- Height, depth, diameter
- Lowest Common Ancestor (LCA)
- Binary Search Trees (insert, delete, search)

**Practice:** 25 problems
- LeetCode: Maximum Depth, Validate BST, Invert Binary Tree, Binary Tree Level Order Traversal, Serialize/Deserialize Tree

***

#### **Week 9: Heaps & Priority Queues**[2]
**Concepts:**
- Min-heap, max-heap
- Heap operations, heapify
- K-th largest/smallest problems
- Top K elements pattern

**Practice:** 10-12 problems
- LeetCode: Kth Largest Element, Top K Frequent Elements, Merge K Sorted Lists

***

#### **Week 10: Graphs**[4][2]
**Concepts:**
- Graph representations (adjacency list/matrix)
- BFS & DFS traversal[5]
- Cycle detection
- Topological sorting
- Connected components
- Shortest path (Dijkstra's, Bellman-Ford)[2]

**Practice:** 25 problems
- LeetCode: Number of Islands, Clone Graph, Course Schedule, Network Delay Time, Word Ladder

***

## **Phase 4: Algorithmic Techniques (Weeks 11-14)**

#### **Week 11-12: Sorting & Searching**[6][2]
**Concepts:**
- Bubble, selection, insertion sort
- **Master:** Merge sort, Quick sort[5]
- Binary search (basic + variations)
- Search in rotated arrays

**Practice:** 15 problems
- LeetCode: Search Insert Position, Find Peak Element, Search in Rotated Sorted Array, Merge Intervals

***

#### **Week 13-14: Dynamic Programming (DP)**[4][5]
**Concepts:**
- 1D DP (Fibonacci, climbing stairs)
- 2D DP (knapsack, longest common subsequence)
- Memoization vs tabulation
- DP on strings, arrays, grids
- State transition, optimal substructure

**Practice:** 30-40 problems (most important!)
- LeetCode: Climbing Stairs, Coin Change, Longest Increasing Subsequence, Edit Distance, House Robber

***

#### **Week 15: Greedy Algorithms**[4]
**Concepts:**
- Greedy choice property
- Activity selection, interval problems

- When greedy works vs DP

**Practice:** 12 problems
- LeetCode: Jump Game, Gas Station, Meeting Rooms II
